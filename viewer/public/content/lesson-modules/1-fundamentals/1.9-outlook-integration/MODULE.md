## Section 1: Email Drafting

Copilot helps you draft emails faster and with the right tone.

### Accessing Copilot in Outlook

**New Email:**
1. Click "New Email"
2. Click the **Copilot** icon in the toolbar
3. Or press **Alt + I** to open Copilot

**Replying:**
1. Open an email
2. Click Reply
3. Click Copilot icon ‚Üí "Draft with Copilot"

### Drafting a New Email

**Basic prompt:**
```
Draft an email to the engineering team about the upcoming 
onboarding wizard launch. Include the launch date (Feb 15),
key features, and ask them to review the final checklist.
```

**With specific tone:**
```
Draft a professional email to the VP of Sales updating them 
on Q1 product roadmap progress. Be concise and focus on 
business impact. 3 paragraphs max.
```

## Section 2: Email Summarization

Long email threads are a PM reality. Copilot summarizes them instantly.

### Summarizing Threads

1. Open a long email thread
2. Click the **Copilot** icon at the top of the email
3. Or look for "Summary by Copilot" automatically displayed

### Summarization Prompts

**Basic summary:**
```
Summarize this email thread
```

**Focused summary:**
```
Summarize this thread focusing on:
1. What decisions were made
2. What's still open
3. What action items exist
```

**Find specific info:**
```
What did Sarah say about the timeline?
```

**Action-focused:**
```
What action items are mentioned in this thread?
Who is responsible for each?
```

## Catching Up on Inbox

When you're behind on email, Copilot helps you triage:

### Inbox Catch-Up Prompts

**In a specific thread:**
```
I haven't read the last 5 messages. What do I need to know?
```

**Finding urgent items:**
```
Is there anything in this thread that requires immediate action from me?
```

**Preparation:**
```
What context do I need to reply to this effectively?
```

## Section 3: Calendar & Meeting Prep

Copilot helps you prepare for meetings from Outlook.

### Meeting Prep

Before a meeting, Copilot can help you prepare:

1. Open the calendar event
2. Click Copilot
3. Ask for preparation help

### Meeting Prep Prompts

**General prep:**
```
Help me prepare for this meeting. What should I know?
```

**Based on emails:**
```
Summarize recent emails related to this meeting topic
```

**Agenda creation:**
```
Based on the meeting invite, suggest an agenda with 
time allocations for a 30-minute meeting
```

**Questions to ask:**
```
What questions should I prepare for this meeting?
```

## Email Templates for PMs

Save these for quick use:

### Stakeholder Update
```
Draft an email to [stakeholder] with a project update:
- Project: [name]
- Status: [on track/at risk/blocked]
- Key updates: [2-3 points]
- Next steps: [1-2 items]
- Ask: [what you need from them]

Tone: Professional, concise
```

### Meeting Request
```
Draft an email requesting a meeting:
- Purpose: [topic]
- Duration: [time needed]
- Urgency: [timeline]
- Proposed times: [suggest flexibility]
- Context: [brief background]
```

### Feedback Request
```
Draft an email asking for feedback on [document/feature]:
- What it is: [brief description]
- What feedback you need: [specific asks]
- Deadline: [when you need it by]
- How to provide feedback: [instructions]
```

### Status Report
```
Draft a weekly status report email:
- Accomplishments: [this week's wins]
- In progress: [current work]
- Blockers: [issues needing help]
- Next week: [upcoming priorities]
- Asks: [what you need]
```
