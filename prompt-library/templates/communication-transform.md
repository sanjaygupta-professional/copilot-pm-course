# Communication Transformation Template

## Multi-Audience Transform

```
I need to communicate this update to three different audiences:

THE UPDATE:
[Your core message/update here]

Please create three versions:

1. **SLACK MESSAGE** (for the product team)
   - Casual, friendly tone
   - 3-4 sentences max
   - Include relevant emoji

2. **EXECUTIVE EMAIL** (for leadership)
   - Professional tone
   - Lead with business impact
   - Include metrics if available
   - 2 short paragraphs

3. **WIKI/NOTION DOC** (for documentation)
   - Detailed and structured
   - Include: Overview, Key Points, Details, Next Steps
   - Use headers and bullet points
```

## Single Transform

```
Rewrite this for [AUDIENCE]:

Original:
[Your text]

Requirements:
- Tone: [casual/professional/technical]
- Length: [brief/moderate/detailed]
- Focus on: [metrics/story/details]
- Format: [prose/bullets/structured]
```

## Stakeholder-Specific

```
Transform this update for [STAKEHOLDER NAME]:

About them:
- Role: [their role]
- They care about: [what matters to them]
- Communication preference: [detailed/brief/visual]

Original message:
[Your message]
```
