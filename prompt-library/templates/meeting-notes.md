# Steering Committee Notes Processing Template

## Basic Prompt

```
Please analyze these steering committee meeting notes and create a structured summary suitable for enterprise governance:

## Executive Summary
- 3-4 sentences summarizing the key outcomes

## Action Items
- List each action item
- Assign owner (based on context)
- Note deadline if mentioned
- Flag priority (Critical/High/Medium)

## Decisions Made
- List any decisions that were finalized
- Note who made/approved the decision

## Risks & Escalations
- Any issues that need to be escalated to sponsors
- Competitive threats mentioned
- Governance concerns raised

## Open Questions
- Items needing follow-up or sponsor input

## Next Steps
- What happens next? When is the next checkpoint?

Format as clean markdown suitable for a steering committee update.
```

## Enhanced Version (with Accenture context)

```
I'm a Senior Product Owner at Accenture's Internal Digital Products team.
Analyze these meeting notes from our [meeting type] meeting.

Context:
- Product: AKX (Accenture Knowledge Exchange) or ADH (Accenture Delivery Hub)
- Sponsor: Richard Thorne (CKO) or Maya Patel (CPO)
- Stakeholders: [relevant names]

Extract:
1. **Executive Summary** (for sponsor brief)
2. **Action Items** (with owners, deadlines, priority)
3. **Decisions** (what was finalized, by whom)
4. **Risks/Escalations** (for steering committee attention)
5. **Open Questions** (needs sponsor input)
6. **My Action Items** (specifically for me as Product Owner)
7. **Next Meeting Prep** (what to prepare for next session)

Format for: [Steering Committee Update / SharePoint / Email to Sponsor]
```

## Quick Action Items Version

```
Extract action items with owners and deadlines from these steering committee notes:
[paste notes]

Flag any items that need executive sponsor attention.
```

## Governance Summary Version

```
Analyze these meeting notes and create a governance summary:

Focus on:
1. Investment decisions made (budget implications)
2. Approval status of initiatives
3. Risk items requiring steering committee attention
4. Timeline commitments made
5. Resource allocation decisions

Format for steering committee records.
```

## Multi-Meeting Synthesis

```
I have notes from 3 meetings this week. Please synthesize them:

Meeting 1: [paste or describe]
Meeting 2: [paste or describe]
Meeting 3: [paste or describe]

Create a consolidated summary with:
1. All action items across meetings (deduplicated)
2. Key decisions made this week
3. Issues requiring escalation
4. Prep needed for next week's steering committee

Organize by priority, not by meeting.
```
